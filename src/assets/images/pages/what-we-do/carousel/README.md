# Carousel Images Folder

Add your work showcase images to this folder and they will automatically appear in the carousel on the "What We Do" page.

## Supported formats:
- .jpg / .jpeg
- .png
- .svg

## Instructions:
1. Add images to this folder
2. Images will automatically be loaded into the carousel
3. The carousel displays 2 images per slide
4. Navigation controls appear automatically when you have more than 2 images

## Naming convention:
- Use descriptive names like: `community-workshop-nepal.jpg`
- Avoid spaces (use hyphens or underscores)
- The filename will be used as the alt text (cleaned up automatically)
